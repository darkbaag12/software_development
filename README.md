# Restaurant Waiting & Table Assignment System

## 프로젝트명
소규모 음식점을 위한 웨이팅 및 테이블 배정 관리 시스템

## 프로젝트 개요
본 프로젝트는 소규모 음식점에서 고객 웨이팅 등록, 고객 호출, 테이블 배정, 입장 처리, 취소 및 노쇼 처리를 관리하는 웹 기반 시스템이다.

이 프로젝트의 목적은 단순한 CRUD 구현이 아니라, 요구사항 분석과 요구 모델링을 먼저 수행한 뒤 이를 바이브코딩의 기준 문서로 활용하는 것이다.

## 구현 범위

### 포함 기능
- 웨이팅 등록
- 대기 목록 조회
- 고객 호출
- 테이블 추천 및 배정
- 입장 처리
- 취소 처리
- 노쇼 처리
- 테이블 상태 관리

### 제외 기능
- 예약 기능
- 실제 SMS 발송
- 실제 로그인/회원가입
- 결제 기능
- 외부 플랫폼 연동
- 복잡한 매출/정산 기능
